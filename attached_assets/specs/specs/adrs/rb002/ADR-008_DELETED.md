# ADR-008: Deliberately Deleted

This ADR was a draft (`DRAFT_ADR-004_Entity_State_Model.md`) that proposed a four-state model for entities. It was superseded by **ADR-009: Decoupled Entity Status Model**, which introduced a clearer, more robust two-axis model (`participation_status` and `verification_status`).

The original file was deleted to prevent confusion, and this placeholder has been created to confirm that its absence is deliberate.
